# Regressão Multipla - Stepwise Backward e Box-Cox em Dados de Doença Renal Crônica

Este projeto apresenta uma análise de regressão múltipla aplicada ao dataset de Chronic Kidney Disease, utilizando Python e bibliotecas estatísticas como statsmodels, scipy e matplotlib.

O objetivo principal foi prever os níveis de creatinina sérica (sc) a partir de variáveis clínicas como idade, pressão arterial, glicose sanguínea, ureia e hemoglobina.

Durante o projeto foram aplicadas:

Regressão Linear Múltipla (OLS);
Seleção de variáveis com Stepwise Backward Elimination;
Análise de resíduos;
Avaliação de métricas como R², R² ajustado, AIC e BIC;
Transformação Box-Cox para melhorar o ajuste do modelo.

Além da implementação do código, o projeto também documenta dificuldades práticas enfrentadas durante a interpretação estatística dos resultados, tornando a análise útil tanto para aprendizado acadêmico quanto para estudos introdutórios em modelagem estatística aplicada à saúde.
